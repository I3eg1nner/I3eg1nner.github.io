Arch Linux In Lenovo 小新 14Pro AMD

Arch Linux In Lenovo 小新 14 Pro AMD

为什么要选择 Arch Linux？

有一定的跟风成分，在公众号看到了苑房弘老师在迁移到 Arch Linux 上，从视觉效果上我觉得尝试一下 Arch Linux，后续折腾虚拟机安装，但是在使用 Hyprland 时总是出现问题（第一天折腾）。刚好朋友也在重装系统，交流的时候，他顺口说了句我的空闲笔记本装 Linux 或许才是最佳选择，第二天就打算尝试用 Lenovo 小新 14 Pro AMD 版来直接单系统 Arch Linux。以下是我在搜集资料的过程中遇到的很有帮助的连接

  • archlinux 简明指南 | archlinux 简明指南 (icekylin.online)
    • 专注于技术，和谐了一部分内容，添加了大量安装的过程中的截图，增加了部分参数和命令的解释以及关于某些错误的处理方法，便于理解和操作
    • 按照文档中的安装过程逐步安装就行（安装前的准备也要看），如果不需要 Hyprland 的话，基本无坑，想要使用 Hyprland 的话需要参考下一个链接里的说明
    • 输入法的问题可以参考第三个链接
    • AMD 集显也需要参考此链接
  • Hyprland on Arch Linux v3 - Nvida, Amd and Intel GPU support - YouTube
    • 安装 Hyprland 的脚本，作者在维护和更新，也可以在 discord 上和作者进行沟通交流
    • 如果想使用这个脚本的话，请注意不要按照上一个教程中那样配置 archlinuxcn，安装过程中可能会遇到许多关于签名的问题，导入信任密钥后可能会遭遇另一些问题
    • 脚本运行后如果有软件安装失败，可以使用 yay 手动安装（建议参考作者在 GitHub 中的 README 文档）
    • 如果登录成功后桌面是黑色的，可能是集显驱动没装，参考第一个链接
    • 作者的壁纸可以在他 GitHub 发布的 v2.1 中找到
    • 常用快捷键汇总（建议 Win + H 查看手册更加详细，而且分类方便查找）：
      • win + D 启动 Wolf，类似于程序菜单
      • win + Q 关闭这个窗口
      • win + Enter 开启 terminal
      • win + shift + W 切换 waybar 图标（界面上方的任务栏）
      • win + B 关闭 Waybar
      • win + alt + W 切换 wayber 的位置
      • win + ctrl + W 指定壁纸切换
      • win + W 切换壁纸
      • win + shift + S 手动截图
      • win + shift + prtsc 手动截图并保存
      • win + alt + V 查看剪切板历史，双击复制
      • win + T 开启文件管理器
    • 窗口调节快捷键
      • win + shift + 空格: 浮动窗口模式
      • win + 鼠标左键: 可拖动窗口
      • win + 鼠标右键: 可调整窗口大小
      • win + 数字: 进入到指定桌面
      • win + shift + 数字: 将窗口扔进某个桌面
      • win + tab 切换桌面
      • win + 空格：多窗口时调整窗口位置（需要鼠标配合）
    • 分辨率调整 ok 后如果希望调整字体大小，需要使用 GTK Setings 中的 font 选项卡
  • ArchLinux下Hyprland配置指北 - 哔哩哔哩 (bilibili.com)
    • 关于输入法，第一个链接中是使用的系统设置，但我的情况只能手动设置，设置方法参考此链接中提到的添加 exec-once=fcitx5 --replace -d 到某个文件中
  • 部分时候你可能需要挂代理
  • wayland下使用腾讯会议分享屏幕 - 知乎 (zhihu.com)
    • 腾讯会议现在已经快成必需品了，这个链接是一种解决腾讯会议在 wayland 中无法共享的方法（显示有问题可以使用 wemeet-x11 ）
    • 你可能需要先使用 sudo pacman -S linux-headers

  • 中文输入法还是有问题
    • 我无能为力，我也面临着这个问题，不过我容忍了它，重启了一次自己好了
    • 建议你参考Fcitx5 官方文档
  • 声音无法控制
    • 我也遇到了，但是多次重启之后，莫名其妙又行了
  • 分辨率问题
    • 调整 Mirrors.conf 即可
